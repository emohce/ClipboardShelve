# src/cpns Boundary

## 负责内容
- 页面组件、列表组件、交互组件、局部状态驱动的界面行为。

## 不负责内容
- 插件运行时初始化。
- 持久化、全局热键注册、窗口级生命周期管理。

## 常见误改点
- 只看组件表象，不确认真实运行通路就反复调局部策略。
- UI 层问题和全局运行时问题混在一起改。
- 在列表滚动、高亮、删除恢复问题上先调阈值，不先确认真实滚动通路。

## 修改前必须确认
- 当前问题属于组件行为、全局行为，还是两者交叉。
- 如果涉及滚动、焦点、高亮，先确认真实滚动祖先和真实焦点入口。
- 是否命中相关错误记录。
