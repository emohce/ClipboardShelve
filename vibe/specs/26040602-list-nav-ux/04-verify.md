# 验证清单

**说明**：以下在 uTools 插件环境手测；本提交未执行 `pnpm run build`。

## 滚动

- [ ] 长列表中部连续 `↑`/`↓`：行已在视口内时不应整列表反复居中抽动。
- [ ] 列表顶/底与 `forceScroll`：首项、末项仍可滚入合理可见区。
- [ ] 在末项按 `↓` 触发 `loadMore` 后仍能选中新段首项（若数据存在）。

## 视觉与多选

- [ ] 非多选：仅当前焦点行有主色左轨与轻外环。
- [ ] 多选：已勾选行有左侧 ✓ 标记与中性质感底色；当前空格焦点行左轨仍明显。
- [ ] 焦点行同时勾选：左轨 + ✓ 同时可见，不依赖同一根 `::before` 条。

## 回归（抽样）

- [ ] 长按方向键连发滚动仍可用。
- [ ] 鼠标悬停预览与键盘导航互不干扰（既有挂起逻辑）。
