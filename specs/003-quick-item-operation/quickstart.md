# Quickstart: 003-quick-item-operation

## 1. 准备

1. 安装依赖（若尚未安装）：`pnpm install`
2. 启动本地开发：`pnpm run serve`
3. 确认当前分支：`003-quick-item-operation`

## 2. 核心回归场景

### 场景 A：`F2` 别名新增/更新

1. 在普通列表选中单条目，按 `F2`。
2. 为无别名条目新增别名；为已有别名条目更新别名。
3. 切到收藏页重复步骤，确认弹层流程仍可用且别名可保存。
4. 在别名弹窗输入任意值后按 `Enter`，确认保存并关闭弹窗。
5. 再次打开别名弹窗，修改输入后按 `Esc`，确认本次修改被取消。

**预期**: 普通页与收藏页都能完成别名新增/更新；收藏页未丢失既有弹层体验；弹窗内 `Enter` 保存、`Esc` 取消，且不会误触发主列表回车快捷键。

### 场景 B：抽屉序号键迁移

1. 选中条目，按 `ctrl+ArrowRight` 展开操作抽屉。
2. 观察第 1~n 项序号。
3. 依次按 `ctrl+alt+1...9`，核对执行结果是否匹配同序号操作。
4. 验证 `ctrl+shift+1...9` 不再作为默认自动执行。

**预期**: 新键位执行与抽屉序号一致，旧键位不再触发默认序号执行。

### 场景 C：`shift+Enter` 别名保存触发

1. 准备“单文件且有别名”条目，按 `shift+Enter`。
2. 准备“组合粘贴”条目，按 `shift+Enter`。
3. 准备“纯文本”条目，按 `shift+Enter`。

**预期**:
- 单文件且有别名：走重命名后粘贴。
- 组合/纯文本：保持原组合粘贴，不进入重命名分支。

### 场景 D：条目别名展示与图片布局

1. 准备带别名的文本条目，观察列表是否“别名优先 + 原始文本省略展示”。
2. 准备带别名的文件条目，观察列表是否“别名优先 + 原始文件摘要省略展示”。
3. 准备带别名的图片条目，观察别名是否显示在图片左侧。
4. 点击图片并悬浮预览，确认预览与点击复制通路未受影响。

**预期**:
- 文本/文件条目展示“别名优先 + 原始信息省略”。
- 图片条目左侧可见别名，图片缩略预览、点击复制、悬浮预览行为保持不变。
