# AI Error Memory

## 目标
- 为所有 AI 提供统一、可检索、可复用的错误复盘知识库。
- 记录“症状 -> 误判链路 -> 已证伪方案 -> 已确认通路 -> 决策规则”。

## 使用方式
1. 先从 [`ai-rules/00-error-memory.md`](../../ai-rules/00-error-memory.md) 检索索引。
2. 命中后阅读对应记录正文。
3. 按记录中的“推荐优先策略”实施。
4. 如果本次发现了新的失败模式，新增一条记录并回填索引。

## 命名规则
- 文件名格式：`YYYY-MM-DD-slug.md`
- 记录编号格式：`EM-YYYY-MM-DD-slug`

## 最低要求
- 只记录可复用、可证伪、对后续任务有约束价值的失败模式。
- 仅有临时猜测但没有形成稳定结论的内容，不入库。
- 每条记录都要写明“禁止再试的做法”和“推荐优先策略”。
