# EzClipboard

多类型剪贴板历史：文本 / 图片 / 文件；收藏与锁定；MD5 去重；多选合并复制与粘贴；全文预览、图片预览、操作抽屉；按时间段清理；存储路径与条数、保存天数、快捷键与主页功能可配置。

**限制**：文件条目暂不写回系统剪贴板；文本与图片可复制回剪贴板。

## 默认操作

| 动作 | 快捷键 |
|------|--------|
| 列表移动 | `↑` `↓` 或 `Ctrl+K` / `Ctrl+J` |
| 复制 | `Enter` |
| 复制并锁定 | `Ctrl+Enter` |
| 收藏 | `Ctrl+S` |
| 锁定 | `Ctrl+U` |
| 预览图/文 | `Shift` |
| 全文预览 | `←` |
| 操作抽屉 | `→` |
| 搜索 | `Ctrl+F` |
| 全部/有锁条件切换后继续搜 | `Ctrl+Shift+U` |
| 收藏标签搜索 | `Ctrl+Alt+F` |
| 多选 | `空格` |
| 清除对话框 | `Shift+Del` 或 `Shift+Backspace` |
| 退出 | `Esc` |
