# docs/VersionDesc 说明

本目录 `docs/VersionDesc/` 用于记录 EzClipboard 的版本演进说明，统一沉淀：

- 当前版本基线说明
- 每次版本更新的增量记录
- 阶段性或正式版本的全量功能记录

## 记录规则

### 1. 每次版本更新必须新增增量记录

建议命名：

- `YYYYMMDD-vX-某版本增量记录.md`

建议内容：

- 版本标识
- 本次目标
- 背景问题
- 方案评估
- 最终采用方案
- 关键改动文件
- 影响范围
- 风险与兼容性说明
- 验证结果
- 遗留问题

### 2. 在阶段性版本结束后更新全量功能记录

建议命名：

- `YYYYMMDD-vX-全量功能记录.md`

建议内容：

- 当前版本全部能力概览
- 各 tab 功能说明
- 搜索规则
- 收藏与标签能力
- 删除与清理规则
- 快捷键能力
- 设置项说明
- 已知限制

### 3. 当前基线文档

- `20260331-v0-当前版本架构与迭代基线说明.md`

后续所有版本改动，默认以这份文档为初始参考基线。
