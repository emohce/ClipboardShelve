# 超级剪贴板 · 简明说明

> 面向使用者，与开发文档 [插件核心要素说明.md](插件核心要素说明.md) 对应。

【核心功能】

- 自动记录：复制过的文字/图片/文件自动进入列表
- 搜索：关键词快速过滤历史记录
- 收藏：重要内容长期保存与归类
- 锁定：关键条目加锁，防止误删
- 多选批量：一次处理多条记录
- 预览：文本/图片快速预览
 
---

【键盘操作】

1. `↑` `↓` / `Tab` 上下移动，`←` `→` 切换 Tab 页
2. `Space` 选择，`Enter` 粘贴，`Delete` 删除
3. `/` 或任意字符进入搜索，`Command/Ctrl+C` 复制
4. `Ctrl+S` 收藏/取消，`Ctrl+U` 锁定/解锁
5. `Ctrl+C` 复制，`Ctrl+Enter` 复制并锁定
6. `→` / `Ctrl+→` 打开抽屉，`Ctrl+←` 查看全文
7. `Alt+1~9` 快速复制第 1~9 项，`Ctrl+Alt+1~9` 执行抽屉第 1~9 项
8. `Ctrl+Delete` 强制删除，`Shift+Delete` 按时间段清理
9. `F2` 编辑别名
