# 超级剪贴板 · 简明说明（用户）

> 面向使用者，与开发文档 [插件核心要素说明.md](插件核心要素说明.md) 对应；只写「你能感知到什么」。

---

## 一句话

- 在 **uTools** 里管理**剪贴板历史**：自动记录**文字 / 图片 / 文件**，支持搜索、收藏、批量与快捷操作；数据保存在本机。

## 打开方式

- 呼出 uTools，输入 **「剪切板」「剪贴板」或「Clipboard」** 进入插件。

## 核心功能（列表）( `ctrl|command` 记作 `c-`)

- **自动记录**：复制过的内容进入列表，按时间查看。
- **搜索**：按关键词过滤历史/收藏（范围以当前页面为准-`c-f`|任意字符）。
- **收藏**：重要内容放入收藏，便于长期保存与归类(`c-s`触发收藏|取消)(`F2`触发收藏的编辑)。
- **锁定**：关键条目加锁，减少误删风险(`c-u`)。
- **复制/粘贴**：把选中条目快速复制回系统剪贴板；部分模式可一键复制并粘贴（以你的系统与设置为准-`enter`|鼠标左键）。
- **多选批量**：一次选多条进行批量处理；混合类型行为以当前版本为准(-`space`)。
- **预览**：文本/图片支持快速预览（必要时可进入更大视图-`shift` `shift+方向键`触发预览下的移动）。
- **快捷键**：主界面/搜索态/抽屉/预览/设置等均有快捷键，**以设置页展示为准**（支持自定义覆盖--暂未实现）。

## 设置建议（列表）

- **存储位置**：放在你常备份的目录，迁移更方便。
- **保存条数/天数**：按使用频率与磁盘情况调整；收藏通常更适合长期保留。
- **主页功能区**：勾选常用功能并排序，提高效率。

## 常见问题（列表）

- **长时间无新记录**：检查系统剪贴板权限、以及插件是否在 uTools 中正常启用。
- **升级后行为变化**：以当前版本界面与市场更新说明为准。
