# 超级剪贴板 · 简明说明

> 面向使用者，与开发文档 [插件核心要素说明.md](插件核心要素说明.md) 对应。

【核心功能】

- 记录：倒序自动记录(使用即刷)
- 预览：文本/图片, 预览滚动
- 收藏：持久化, 标签, 备注
- 加锁: 短期防删
- 别名: 增强区分, 以别名粘贴 
- 搜索：过滤(文字|描述|备注|别名), 有锁|全部
 
---

【键盘操作】

* `↑` `↓` / `Tab` 上下移动，`←` `→` 切换 Tab 页
* `␣` 多选，`cr` 粘贴，`Del` 删除, `c-cr` 粘贴并加锁, `c-del`|`a-del` 加锁也删
* `c-s` 收藏/取消，`c-u` 锁定/解锁, `c-s-u` 搜索-有锁|全部
* `c-→` 打开抽屉，`c-←` 查看全文 (相反方向键|Esc取消)
* `a-1~9` 快速复制第 1~9 项，`a-1~9` 执行抽屉第 1~9 项
* `F2` 编辑别名(收藏tab区分), 单文件`s-cr`以别名粘贴
