# 超级剪贴板 · 简明说明（用户）

> 面向使用者，与开发文档 [插件核心要素说明.md](插件核心要素说明.md) 对应；只写「你能感知到什么」。

---

## 一句话

- 在 **uTools** 里管理**剪贴板历史**：自动记录**文字 / 图片 / 文件**，支持搜索、收藏、批量与快捷操作；数据保存在本机。

## 打开方式

- 呼出 uTools，输入 **「剪切板」「剪贴板」或「Clipboard」** 进入插件。

## 核心功能（列表）( `ctrl|command` 记作 `c-`；`shift` 记作 `s-`；`shift+Enter` 记作 `s-cr` )

- **自动记录**：复制过的内容进入列表，按时间查看。
- **搜索**：按关键词过滤历史/收藏（范围以当前页面为准）。
- **收藏**：重要内容放入收藏，便于长期保存与归类（`c-s` 收藏或取消）。
- **锁定**：关键条目加锁，减少误删风险（`c-u`）。
- **复制/粘贴**：把选中条目快速复制回系统剪贴板；部分模式可一键复制并粘贴（以你的系统与设置为准-`enter`|鼠标左键）。
- **多选批量**：一次选多条进行批量处理；混合类型行为以当前版本为准（`space`）。
- **预览**：文本/图片支持快速预览（必要时可进入更大视图）。
- **快捷键**：主界面/搜索态/抽屉/预览/设置等均有快捷键，**以设置页展示为准**（支持自定义覆盖--暂未实现）；操作抽屉展开后的数字执行等见下文分组说明。

## 主界面常用快捷键（按功能分组）

下文 **`c-a-`** 表示 **`ctrl+alt`**；**`a-1..9`** 表示 **`alt+1`…`alt+9`**（与 `c-` / `s-` / `s-cr` 约定同段首）。

- **搜索与筛选**：`c-f`，任意可打印字符唤起搜索，`c-s-u` 切换「全部 / 有锁」条件后继续搜索。
- **收藏与标签**：`c-s` 收藏/取消，`c-u` 锁定/解锁；`c-a-f`、`c-s-f` 打开收藏标签搜索；在收藏主 tab 下可用 `c-Tab`、`c-s-Tab` 切换收藏子 tab。
- **别名与按别名粘贴**：`F2` — **未收藏**条目为别名新增/更新，**已收藏**条目为标签/备注编辑弹层（由是否已收藏决定，不限于「收藏」tab）；`s-cr` 在单条目且含别名等条件满足时按别名保存或按别名重命名后粘贴（文件/图片等以当前版本为准）。
- **列表导航与类型页**：`↑`/`↓` 与 `c-k`/`c-j` 上下移动；`PgUp`/`PgDn` 与 `c-↑`/`c-↓` 翻页；`←`/`→` 切换类型页；`Tab`/`s-Tab` 切换顶部类型 tab。
- **复制与多选**：`Enter` 复制（及部分场景粘贴）、`c-Enter` 复制并锁定或多选合并粘贴；`c-c` 复制；`space` 向下多选；`a-1..9` 快速复制列表中第 1～9 项。
- **抽屉与序号**：`c-→` 打开操作抽屉；抽屉展开后 `c-a-1..9` 执行菜单第 1～9 项（替代旧版 `c-s-`+数字）。
- **全文与预览**：`c-←`、`s-F2` 查看全部/全文；按住 `Shift` 预览图片；`s-` 与方向键在预览区域内滚动。
- **删除与清理记录**：`Del`/`Backspace` 删除选中；`c-Del`/`c-Backspace` 强制删除（含锁定项）；`s-Del`/`s-Backspace` 打开按时间段清理。
- **其它**：`Escape` 关闭对话框或退出设置等；`c-alts` 打开设置；`c-1..9` 切换到第 1～9 个顶部标签页。

## 设置建议（列表）

- **存储位置**：放在你常备份的目录，迁移更方便。
- **保存条数/天数**：按使用频率与磁盘情况调整；收藏通常更适合长期保留。
- **主页功能区**：勾选常用功能并排序，提高效率。

## 常见问题（列表）

- **长时间无新记录**：检查系统剪贴板权限、以及插件是否在 uTools 中正常启用。
- **升级后行为变化**：以当前版本界面与市场更新说明为准。
