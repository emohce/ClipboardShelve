# preview 原始需求

## 优化

* 预览图片, 上下居中展示, 现在的占据比例不用改, 
  * 图片预览时: 高度小要自动上下居中展示; 宽度小则自动扩放并直至按键滚动预览; 能铺满尽量以单页预览为主, 可以shift+左右键移动(非缩放)
  * 高度很高时, 以宽度铺满居中对齐为准, 支持shift+上下键进行滚动, 记住不是缩放
* 预览文字
  * 文字单行的展示效果类似图片, 尽量铺满窗口90%宽度, 上下居中展示, 单行展示不全时 自动单行使用缩略符(` ...` ), 不要自动换行
  * 很多行展示时, 支持shift+上下键进行滚动, 不是缩放
* 预览状态下
  * 已经时预览时, 阻断鼠标移动导致的预览切换, 只有松开shift才可以切换预览的元素
